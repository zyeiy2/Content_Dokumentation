# ChatGPT für Profis

Willkommen zur Profischulung für **ChatGPT**!  
Diese Seite richtet sich an erfahrene Nutzer:innen, die das volle Potenzial von ChatGPT ausschöpfen und in komplexere Workflows integrieren möchten.

---

## 🤖 Was steckt unter der Haube?

**ChatGPT** basiert auf einem leistungsfähigen Sprachmodell (GPT-4), das kontextbasierte Texte versteht, generiert und optimiert.  
Als Profi kannst du mit mehrstufigen Prompts, Rollen, Formatvorlagen und sogar technischen Schnittstellen (z. B. API, Plugins) arbeiten.

---

## 🎯 Wofür kannst du ChatGPT als Profi nutzen?

| Aufgabe | Beispiel |
|--------|---------|
| Code-Refactoring | „Verbessere diesen Python-Code unter Berücksichtigung der SOLID-Prinzipien.“ |
| Test-Case-Generierung | „Erstelle Unit-Tests für folgende Funktion in Python.“ |
| Rollenbasierte Szenarien | „Du bist ein Data Scientist. Erkläre DAX vs. M in Power BI für ein Managementbriefing.“ |
| Datenanalyse & Modellierung | „Welche Schritte brauche ich für eine Segmentierung mit k-Means in Python?“ |
| Prompt-Design & Automatisierung | „Formuliere einen Prompt, der wiederverwendbar für LinkedIn-Posts im HR-Bereich ist.“ |

---

## 🛠️ Nächste Schritte im Workflow

### 1. Rollen und Ziele kombinieren

```text
Du bist ein erfahrener Backend-Entwickler. Bitte analysiere diesen Code auf Schwachstellen in Bezug auf Skalierbarkeit:
[CODE HIER EINSETZEN]
````

### 2. Kontext mitliefern

```text
Ich arbeite an einer Azure Data Factory Pipeline. Ziel ist es, inkrementelle Ladeprozesse zu bauen. Gib mir Best Practices.
```

### 3. Iterativ optimieren

```text
Erweitere den Code so, dass Fehler beim API-Call abgefangen und geloggt werden. Nutze dabei ein Retry-Konzept.
```

---

## 💬 Effektive Profi-Prompts schreiben

### ✅ So funktioniert’s optimal:

* **Arbeite mit Modulen:**
  *Prompt für Recherche → Prompt für Textentwurf → Prompt für Feinschliff*

* **Definiere klare Regeln:**
  *„Bitte verwende maximal 3 Bulletpoints, sprich das Management direkt an und vermeide Fachjargon.“*

* **Schreibe anpassbare Vorlagen:**
  *„Erstelle ein Prompt-Template für interne Projektvorstellungen, anpassbar für verschiedene Zielgruppen.“*

---

## 🧪 Beispiele zum Ausprobieren

```text
Du bist ein agiler Coach. Erstelle einen 60-minütigen Workshop zur Einführung von Kanban für ein IT-Team.

Optimiere folgende SQL-Abfrage für Performance auf einem PostgreSQL-Datenbankcluster.

Fasse diesen Fachtext aus dem Bereich Data Governance für eine PowerPoint-Präsentation zusammen – maximal 4 Slides, Stichpunkte.

Schreibe ein PowerShell-Skript, das alle Dateien eines Verzeichnisses mit Änderungsdatum auflistet und in eine CSV exportiert.

```
